C++ Exam Collection

This project is a simple web application created by the Academics Team of the Evangelical Christian Students Fellowship at Arba Minch University.
It provides previous-year C++ exam questions for junior Pre-Engineering students.
The application also features a Bible verse to inspire students as they study.

Features

Organized collection of past C++ exams

Simple, lightweight web interface

Mobile-friendly layout

Purpose

This project aims to support junior students academically while also encouraging them spiritually. It provides easy access to exam materials gathered from previous batches.

Technologies Used

HTML

CSS
